根据hexo && github 搭建自己的博客

##Github Pages
Github Pages 原本是用于托管在github上的项目的，由于它提供免费稳定的空间，所以可以用来搭建博客。

每个github帐号只能有一个仓库来存放自己的个人主页，而且仓库的名字必须是username/username.github.io,这个特殊的命名约定，让别人可以通过http://username.github.io来访问个人主页。

需要注意的是，个人网页的网站内容必须是在master分支下的。

##部署Hexo到Github Pages

意思就是把Hexo本地生成的博客部署到github上，然后可以通过http://username.github.io来访问博客的内容

首先，需要在自己的本地建立一个与github相连接的项目，然后创建一个archive文件夹。
$ git clone git@github.com:username/username.github.io.git
/username.github.io$ mkdir archive

然后下载安装Hexo(需要提前安装好node.js等插件)，并根据Hexo生成博客模板
/username.github.io$hexo archive

在archive文件中的 _config.xml末尾添加：
deploy:
type: git
repo: git@github.com:username/username.github.io.git
branch: master

然后执行：
/username.github.io$ hexo d
完成部署。

最后，将自己的本地项目推送到github上，然后打开http://username.github.io即可。
